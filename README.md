# Power-BI-Airbnb-Price-Scenario-Analysis-Dashboard
This project showcases an interactive Airbnb Price &amp; Scenario Analysis Dashboard built in Power BI. It analyzes listing prices, availability, occupancy scenarios, and projected revenue using dynamic What-If parameters to help optimize pricing and maximize revenue.
